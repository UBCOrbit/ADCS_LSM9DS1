Detailed Description

Multi-Buffered Serial Peripheral Interface Module.

The MibSPI/MibSPIP is a high-speed synchronous serial input/output port that allows a serial bit stream of programmed length (2 to 16 bits) to be shifted in and out of the device at a programmed bit-transfer rate. The MibSPI has a programmable buffer memory that enables programmed transmission to be completed without CPU intervention
